# Utility Catalog

A detailed technical breakdown of the tools current integrated into AIO Utilities.

## 📝 Grammar Checker
*   **Tech Stack**: LanguageTool Plus API, React
*   **Key Features**: Picky-level grammar check, Interactive suggestions, Smart Fix-all.
*   **Category**: Documents

## 🎂 Age Calculator
*   **Tech Stack**: Vanilla Javascript `Date` logic
*   **Key Features**: Birthday calculation, Date Gap analysis, Persistence via Dexie.
*   **Category**: Productivity

## 📏 Unit Converter
*   **Tech Stack**: Constant-based conversion maps
*   **Key Features**: Mass, Length, Volume, Temperature, Area. Automatic magnitude sorting.
*   **Category**: Math / Engineering

## 🌍 World Clock
*   **Tech Stack**: `Intl.DateTimeFormat` (Native JS)
*   **Key Features**: Global timezone conversion, Persistent watchlists, Day/Night indicators.
*   **Category**: Productivity

## 🔓 PDF Password Remover
*   **Tech Stack**: `pdf-lib`, `@pdfsmaller/pdf-decrypt`
*   **Key Features**: 100% Client-side decryption, Fast processing.
*   **Category**: Documents

## 🖼️ Image Tools
*   **Tech Stack**: HTML5 Canvas API
*   **Key Features**: WEBP/JPG/PNG/SVG conversion, Target-size compression.
*   **Category**: Media

## 🔗 QR Code Generator
*   **Tech Stack**: `qrcode` (NPM)
*   **Key Features**: Dynamic colors, Logo overlays (Center/Corners), SVG/PNG export.
*   **Category**: Generators
